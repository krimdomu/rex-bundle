=pod

rex-bundle is a R?ex module to install needed perl modules into a private folder seperated from the system librarys.

=head2 Dependencies

=over 4

=item *

L<YAML>

=item *

L<LWP::Simple>

=item *

curl

=back

=head2 Usage

In your Rexfile use the following commands.

 install_to 'vendor/perl'
  
 desk "Check and install dependencies";
 task "deps", sub {
    mod "Mod1";
    mod "Foo::Bar";
    # ...
 };

And in your script you can now use these modules with:

 use lib 'vendor/perl';
